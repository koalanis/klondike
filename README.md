# klondike

a simple terminal soilitare
written in nim

```
=======================================
            _    .  ,   .           .
        *  / \_ *  / \_      _  *         
         /    \  /    \,   ((        .    
    .   /\/\  /\/ :' __ \_  `  ))  *    
       /    \/  \  _/ ^  \  ))      *   
     /\/\  /\/\:~      \ |\  *       
    /    \/    --  __   \| \/ \  
   /\/\  /\/\   /  \  | / \/   \
  /    \/    \  /    ||    \   /\
/\/\  /\/\  /\/\  /\/  /\  /\ /  \
=======================================
```

## how to play

```
============================
[Q♤]|24

░░░░ ░░░░ ░░░░ ░░░░
 F0   F1   F2   F3

                        [==]
                    [==][==]
                    [==][==]
    [==]    [==][==][==][==]
    [4♧][==][==][==][==][==]
    [3♢][==][==][==][==][==]
    [2♤][A♤][6♢][5♧][7♡][3♤]
 T0  T1  T2  T3  T4  T5  T6
============================
```


## how to build
```sh
nimble build
```
